# 🚢 Shipping Container Tracking System

A menu-driven C++ console application for managing and tracking shipping containers across global logistics operations.

> **Case Study 87** | ITM Skills University | B.Tech CSE 2025–29 | Semester II

---

## 📌 About

This project simulates a real-world container tracking system where users can register shipping containers, update their locations, search records, generate status reports, and manage data — all persisted to a local text file.

Built as part of the C++ Programming case study to demonstrate **Object-Oriented Programming**, **file handling**, and **modular design**.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Register Container** | Add new containers with unique ID, origin, destination, weight & status |
| **Update Location** | Modify current location and status of any container |
| **Search by ID** | Instantly find and display container details |
| **Display All** | View all records in a formatted table |
| **Status Report** | Category-wise summary with percentages |
| **Delete Container** | Remove records with confirmation prompt |
| **Data Persistence** | All data saved to `containers.txt` automatically |

---

## 🛠️ Tech Stack

- **Language:** C++ (C++11)
- **Libraries:** `<iostream>`, `<fstream>`, `<sstream>`, `<vector>`, `<string>`, `<iomanip>`, `<limits>`
- **Storage:** Text file with pipe-delimited format
- **Compiler:** g++ / MSVC / Clang
